
# Customer Club

In order to persue its cusomers, eash business needs to have a customer club. A club that each customer could earn points and use them to get many benefits as discounts, cash-back or even special gifts and services.

This project is going to be an open source free to use dotnet based customer club.




## Tech Stack

**Client:** Blazor, MAUI

**Server:** Dotnet, Elasticsearch, MediatR, FluentResult, EntityFramework


## Features

- Light/dark mode toggle
- Cross platform
- Multitenant
